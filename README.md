## 其他
> gitbook的操作

根据`SUMMARY.md`来构建目录结构
````bash
	gitbook init
````
根据当前的目录生成静态html
````bash
	gitbook build
````
启动本地的`gitbook`服务
````bash
	gitbook serve
````
